#Cấu trúc của trang HTML
<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8" />
    <title>TITLE</title>
  </head>
  <body></body>
</html>
#CẤU TẠO CỦA THẺ HTML
Cấu tạo thẻ đủ:

<tenthe thuoctinh1="trị thuộc tính 1" ...>Nội dung</tenthe>
Cấu tạo thẻ rỗng:

<tenthe thuoctinh1="trị thuộc tính 1" ... />
